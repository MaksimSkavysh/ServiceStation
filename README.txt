Чтобы запустить приложение необходимы:
apache tomcat local server
MySQL

Инструкция по запуску:
1) указать tomcat, как Run/Debug configuration
2)  localhost:8080
    Application context: /serviceStation
3) запустить(Run) приложение

Валидацию сделать не успел, так как к сожалению по личным обсатаятельствам(переезд) писать пришлось за три с небольшим дня
